# sinister

## Introduction

SPG(single page generator) is a single static page generator with Go lang

## Installation

```
    go get github.com/crispgm/go-spg
```

## Getting Started

```
   ./spg hello-world.md -output=hello.html -var-title="Hello, World" -var-footer="Thanks"
```
